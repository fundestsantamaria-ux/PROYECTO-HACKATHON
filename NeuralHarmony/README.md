Este proyecto consiste en un sistema que recomienda música según el estado emocional de
una persona. Utiliza análisis de texto sencillo para identificar emociones a partir de frases
escritas libremente por el usuario y se conecta con la API de Spotify para sugerir canciones
reales que se ajustan a cómo la persona se siente en ese momento.
La idea principal del proyecto es aprovechar la música como una herramienta para
acompañar y regular las emociones, ayudando al usuario a sentirse comprendido y
acompañado. El sistema permite que la persona escriba frases naturales, no solo una palabra
exacta, y detecta emociones como felicidad, tristeza, enojo, ansiedad, calma o cansancio.
Una vez que la emoción es identificada, el sistema busca automáticamente en Spotify
canciones en español relacionadas con ese estado de ánimo y muestra una lista con
recomendaciones, incluyendo el nombre de la canción, el artista, la portada del álbum y un
enlace directo para escucharlas. Además, el usuario puede ingresar nuevas emociones sin
necesidad de volver a ejecutar el programa, haciendo la experiencia más fluida y amigable.
